Artificial Intelligence

DialogFlow

I recently took up a project that uses DialogFlow to provide automatic tech support for a software. DialogFlow uses Google's Machine Learning network to train its agents. When setting up the chatbot, I had to script the creation of various intents (or questions) that could be asked. Each intent is mapped with a response that the bot provides. Fascinatingly, it takes only a few minutes to train the chatbot using DialogFlow, once all the intents have been uploaded.
